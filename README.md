# Trading Bot — Спекулятивная торговля на MOEX

## Идея
Автоматизированный бот для торговли на российском фондовом рынке через **Tinkoff Invest API**.
Бот получает рыночные данные, генерирует торговые сигналы по заданным стратегиям и отображает состояние портфеля в дашборде.

## Стек
- **Язык:** Python
- **Брокер:** Tinkoff Invest API
- **Дашборд:** Streamlit / Dash (в процессе выбора)
- **Хранилище:** SQLite / DuckDB (локальная история котировок)

## Структура
```
trading-bot/
├── bot/
│   ├── broker/       # Обёртка над Tinkoff API
│   ├── data/         # Получение и хранение котировок
│   ├── strategies/   # Торговые стратегии и сигналы
│   ├── risk/         # Риск-менеджмент (лимиты, стоп-лоссы)
│   └── dashboard/    # Веб-дашборд
└── tests/
```

## Архитектурный принцип
Стратегии не зависят от брокера — они работают только с абстракцией `Signal (BUY/SELL/HOLD)`.
Все ордера проходят через `risk/manager.py` перед исполнением.
